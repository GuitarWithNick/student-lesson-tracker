    Import Spreadsheet

    Upload a CSV spreadsheet to bulk add material, goals, songs, and students. Material rows with a blank category will be placed in Songs.

    Need a starter file? Download import template or download category example

    Sync Material Folder

    Choose a local folder to add its files into System Practice Material. This version adds new material and skips duplicates. It does not remove tracker items if files disappear from disk.

    Best results: choose a parent folder that contains category folders like Songs, Riffs, Technique, or Bass. Nested folders under Songs are ignored cleanly, so Songs/Artist/Album/file.pdf still imports into Songs. For nested groups like Bass / Scales, keep Bass somewhere in the selected path so the app can map it correctly.
